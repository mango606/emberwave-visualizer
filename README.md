# Emberwave

**불멍하듯이 감상할 수 있는 LED 오디오 비주얼라이저**

React · Vite · Web Audio API · Canvas · Tailwind CSS

</div>

## 소개

Emberwave는 브라우저에서 동작하는 오디오 비주얼라이저입니다. 좋아하는 음악을 올리면 실시간 주파수 분석으로 레트로 감성의 LED 바가 음악에 맞춰 살아 움직이고, 잉걸불처럼 번지는 화면을 바라보며 불멍하듯 편안하게 쉬어갈 수 있습니다. 빗소리나 모닥불 같은 유튜브 영상을 배경으로 함께 틀어 나만의 사운드스케이프를 완성할 수도 있습니다.

## 주요 기능

- **음악 재생목록**: 파일을 여러 개 추가하거나 폴더를 통째로 연결하면 폴더 안의 음악이 자동으로 목록에 올라옵니다. 연결된 폴더는 주기적으로 확인해 새로 넣은 곡도 알아서 목록에 반영합니다(Chrome, Edge). 셔플, 드래그로 순서 변경, 삭제, 전체 비우기, 트랙별 재생시간 표시를 지원합니다.
- **레트로 LED 이퀄라이저**: Canvas로 그리는 직사각형 LED 바. 피크 홀드, 네온 글로우, 상하 미러 효과를 담았습니다.
- **18가지 색상 팔레트**: 네온 그린, 신스웨이브 퍼플, 앰버, 오로라 블루, 라벤더, 민트 등.
- **10가지 반응 모드**: 차분한, 신나는, 레트로 8비트, 펄스, 스펙트럼 등 분위기에 맞춰 움직임이 달라집니다.
- **유튜브 배경 재생**: 자연의 소리 영상을 반복 재생하고, 음악과 유튜브 볼륨을 따로 조절합니다.
- **음질 조정**: 베이스 부스트, 보컬 강조 같은 프리셋과 저음·중음·고음 3밴드 슬라이더로 취향에 맞는 소리를 만듭니다.
- **이어지는 감상**: 색상, 모드, 볼륨, 음질 설정은 물론 재생목록까지 브라우저에 저장되어, 창을 닫았다 다시 열어도 그대로 이어집니다.
- **앰비언트 전체화면**: 비주얼라이저 우측 하단에 마우스를 올리면 나타나는 버튼으로, 컨트롤 없이 화면 가득 감상할 수 있습니다.
- **잠금화면 컨트롤**: 모바일에서 화면을 꺼도 재생이 유지되고, 잠금화면에서 곡 정보 확인과 재생, 곡 이동이 가능합니다.
- **라이브 입력**: 마이크 소리나 탭 오디오 공유로도 바가 반응합니다. 탭 소리를 공유하면 유튜브 사운드까지 시각화됩니다.

## 사용 방법

1. 오른쪽 패널에서 파일을 추가하거나 폴더를 연결해 재생목록을 만듭니다.
2. 색상 팔레트와 반응 모드를 골라 원하는 분위기를 맞춥니다.
3. 원한다면 유튜브 링크를 입력해 배경음을 함께 재생하고, 볼륨 믹서로 비율을 조절합니다.
4. 마이크나 탭 소리를 켜면 스피커 밖의 소리에도 LED 바가 반응합니다.
