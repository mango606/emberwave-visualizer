## 소개

Emberwave는 브라우저에서 동작하는 오디오 비주얼라이저입니다. 좋아하는 음악을 올리면 실시간 주파수 분석으로 레트로 감성의 LED 바가 음악에 맞춰 살아 움직이고, 잉걸불처럼 번지는 화면을 바라보며 불멍하듯 편안하게 쉬어갈 수 있습니다. 빗소리나 모닥불 같은 유튜브 영상을 배경으로 함께 틀어 나만의 사운드스케이프를 완성할 수도 있습니다.

## 주요 기능

- **음악 재생목록** — 여러 MP3를 한 번에 올려 순서대로 감상합니다. 셔플, 드래그로 순서 변경, 삭제, 트랙별 재생시간 표시를 지원합니다.
- **레트로 LED 이퀄라이저** — Canvas로 그리는 직사각형 LED 바. 피크 홀드, 네온 글로우, 상하 미러 효과를 담았습니다.
- **12가지 색상 팔레트** — 네온 그린, 신스웨이브 퍼플, 앰버, 오로라 블루 등.
- **10가지 반응 모드** — 차분한, 신나는, 레트로 8비트, 펄스, 스펙트럼 등 분위기에 맞춰 움직임이 달라집니다.
- **유튜브 배경 재생** — 자연의 소리 영상을 반복 재생하고, 음악과 유튜브 볼륨을 따로 조절합니다.

## 사용 방법

1. 오른쪽 패널에서 MP3 파일을 여러 개 선택해 재생목록을 만듭니다.
2. 색상 팔레트와 반응 모드를 골라 원하는 분위기를 맞춥니다.
3. 원한다면 유튜브 링크를 입력해 배경음을 함께 재생하고, 볼륨 믹서로 비율을 조절합니다.
