## 소개

**Emberwave**는 브라우저에서 동작하는 오디오 비주얼라이저입니다. 좋아하는 음악을 올리면 실시간 주파수 분석을 통해 레트로 오디오 기기 감성의 직사각형 LED 바가 움직이고, 여기에 빗소리·모닥불 같은 유튜브 ASMR을 원하는 비율로 섞어 나만의 '불멍' 사운드스케이프를 만들 수 있습니다.

모든 오디오 처리는 사용자의 브라우저 안에서만 이루어집니다. 업로드한 음악 파일은 어떤 서버로도 전송되거나 저장되지 않습니다.

## 주요 기능

- **로컬 MP3 재생 & 재생목록** — 여러 파일을 한 번에 올려 목록 순서대로 이어 재생합니다. 이전/다음 곡 이동, 곡 종료 시 자동 넘김(순환), 탐색을 지원합니다.
- **온전한 로컬 처리** — Web Audio API 기반 실시간 주파수 분석. 음악 파일은 브라우저 밖으로 나가지 않습니다.
- **레트로 LED 이퀄라이저** — Canvas 2D로 렌더링되는 직사각형 세그먼트 LED 바. 피크 홀드, 네온 글로우, 상하 미러 효과를 포함합니다.
- **12가지 색상 팔레트** — 네온 그린, 신스웨이브 퍼플, 앰버, 오로라 블루, 클래식 VU 등.
- **10가지 반응 모드** — 차분한 · 신나는 · 레트로 8비트 · 웨이브 · 펄스 · 스펙트럼 등, 음악 분위기에 맞춰 바의 움직임 스타일을 전환합니다.
- **유튜브 ASMR 믹싱** — 자연의 소리 영상을 백그라운드로 재생하고, 음악과 ASMR 볼륨을 각각 독립적으로 조절합니다.
- **출처 크레딧 표기** — 재생 중인 ASMR 원본 영상의 제목과 링크를 화면에 표시합니다.
- **한 화면 몰입형 UI** — 다크 모드 기반, 스크롤 없이 뷰포트에 딱 맞는 레이아웃.

## 사용 방법

1. 화면 우측 패널에서 **MP3 파일(여러 개 가능)**을 선택하면 재생목록이 만들어집니다. 재생 버튼으로 순서대로 감상하세요.
2. **색상 팔레트**와 **반응 모드**를 골라 원하는 분위기를 만듭니다.
3. (선택) 빗소리·모닥불 등 **유튜브 링크**를 입력해 배경 ASMR을 재생합니다.
4. **볼륨 믹서**로 음악과 ASMR의 비율을 조절해 나만의 사운드를 완성합니다.

> 브라우저 오토플레이 정책에 따라, 오디오는 재생 버튼 클릭 등 사용자 조작 이후에 시작됩니다.

## 기술 스택

| 영역 | 사용 기술 |
| --- | --- |
| 프레임워크 | React 18, Vite 5 |
| 스타일 | Tailwind CSS 3 |
| 오디오 | Web Audio API (`AnalyserNode`, `GainNode`) |
| 렌더링 | Canvas 2D (`requestAnimationFrame`) |
| 외부 연동 | YouTube IFrame API (`react-youtube`) |
| 배포 | Vercel |

## 로컬 개발

```bash
npm install     # 의존성 설치
npm run dev     # 개발 서버 실행 (http://localhost:5173)
```

## 빌드 & 배포

```bash
npm run build   # 프로덕션 번들 생성 (dist/)
npm run preview # 빌드 결과 미리보기
```

Vercel에 배포할 경우, 저장소를 연결하면 프레임워크가 **Vite**로 자동 감지됩니다. Build Command(`vite build`)와 Output Directory(`dist`)가 기본값으로 설정되며, SPA 라우팅을 위한 `vercel.json`이 포함되어 있어 추가 설정 없이 배포됩니다.

## 브라우저 지원

Web Audio API와 Canvas를 지원하는 최신 데스크톱·모바일 브라우저(Chrome, Edge, Safari, Firefox)에서 동작합니다.
